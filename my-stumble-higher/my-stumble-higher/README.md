# Stumble Higher

A curated discovery platform for inspiring content that helps you think differently, grow, and go higher.

## Features

- One-click exploration of curated resources
- Categories spanning books, essays, videos, and tools
- Seamless FramesV2 integration
- Share functionality for spreading inspiration

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- Next.js
- React
- Tailwind CSS
- FramesV2 Integration

## Project Structure

```
stumble-higher/
├── components/         # React components
├── pages/             # Next.js pages
├── public/            # Static assets
├── styles/           # Global styles
└── utils/            # Helper functions
```

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_API_URL=your_api_url
```

## Deployment

Deploy using Vercel:

```bash
vercel
```
